TypeFace

JetBrains Mono

The letters in this typeface are shaped more like rectangles than circles. This makes the text easier for your eyes to follow because the shapes are consistent and clear.

The straight outer edges also create a smooth path for your eyes as you read down the page, without any extra visual distractions.

Although each character keeps the same width, the lowercase letters are made taller. This keeps code lines at the familiar length developers expect, while also improving clarity because each letter has more pixels and appears sharper on the screen.
